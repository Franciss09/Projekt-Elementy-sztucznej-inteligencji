# Projekt-Elementy-sztucznej-inteligencji
Tworzenie sieci neuronowej do przewidywania cen złota

Projekt: Sztuczne Sieci Neuronowe
Część 1: Problem klasyfikacji (Przewidywanie kierunku ceny złota)
W pierwszej części projektu I postanowiliśmy sprawdzić, czy sieć neuronowa poradzi sobie z przewidywaniem ceny złota. Model miał za zadanie przewidzieć, czy w kolejnym dniu cena złota wzrośnie (klasa 1), czy spadnie (klasa 0).

Część 2: Problem regresji (Przewidywanie dokładnej ceny złota)
W drugiej części projektu I zadanie sieci neuronowej było znacznie trudniejsze, miał przewidzieć dokładne oszacowanie konkretnej wartości – ceny złota na dany dzień.

Projekt: Uczenie maszynowe
W projekcie wykorzystano cztery popularne algorytmy dostępne w bibliotece scikit-learn.
